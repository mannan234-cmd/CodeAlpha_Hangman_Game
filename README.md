#  Hangman Game in Python

A simple command-line Hangman game developed in Python as part of the **CodeAlpha Python Programming Internship**.



# Project Overview

Hangman is a word guessing game where the player tries to guess a hidden word one letter at a time.

The game randomly selects a word from a predefined list of five words. The player has a maximum of **6 incorrect guesses** before the game ends.



# Features

- Random word selection
- Predefined list of 5 words
- Input validation
- Tracks guessed letters
- Prevents duplicate guesses
- Win and Lose conditions
- Quit option (`quit`)
- Beginner-friendly Python code

# How to Play

- A random word is selected.
- Guess one letter at a time.
- You have **6 incorrect guesses**.
- Guess the complete word before your chances are over.
- Type **quit** to exit the game.
